# Componentes e Utilitários Compartilhados (Shared)

Este diretório armazena todos os elementos reutilizáveis que são consumidos por mais de um módulo do sistema (ex: usados tanto no painel do Aluno quanto no do RH).

## O que vai aqui:
- `/components`: Elementos genéricos de interface (botões, modais, inputs estilizados, cabeçalhos comuns, wrappers de layouts).
- `/hooks`: Hooks React compartilhados (ex: `useLocalStorage`, `useDebounce`, etc.).
- `/utils`: Funções utilitárias puras (formatação de data, conversão de tempo, formatação de dados textuais).
- `/context`: Provedores de estados globais (ex: tema, notificações).
