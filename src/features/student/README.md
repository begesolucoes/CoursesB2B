# Módulo do Aluno (Colaborador)

Este diretório contém os componentes, layouts e hooks específicos da experiência de consumo dos cursos pelo colaborador final.

## Componentes a serem criados aqui:
- `StudentDashboard.tsx`: Dashboard contendo as métricas de progresso do aluno e abas de cursos ("Não Iniciados", "Em Progresso", "Concluídos").
- `CourseCard.tsx`: Card visual exibindo capa do curso, progresso em porcentagem, data limite e botão de ação.
- `CourseDetailsView.tsx`: Tela contendo a ficha técnica do treinamento selecionado, descrição, carga horária e histórico de tentativas passadas.
